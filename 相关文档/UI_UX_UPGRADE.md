| 维度               | 当前情况                             | 问题等级                  |
| ------------------ | ------------------------------------ | ------------------------- |

## 二、设计方案

### 1. 🎯 品牌色升级

保留绿色基调，增强层次体系：

```
Primary:    #26c99a ──── 主品牌色（运动/健康联想）
Primary+1:  #1fb089 ──── hover 态
Primary+2:  #1a9a75 ──── active 态
Primary-1:  #f0faf6 ──── 浅色背景
PrimaryBorder: rgba(38,201,154,0.25)

Secondary:  #5b8def ──── 好友系统蓝色
Accent:     #f0a030 ──── 高亮/警告
Error:      #e74c3c ──── 错误

Gradients:
  绿色: #26c99a → #1a9a75
  蓝色: #5b8def → #7c6cf0
```

### 2. 🔤 字体体系

| 层级    | 字体                       | 大小     | 字重 | 用途                 |
| ------- | -------------------------- | -------- | ---- | -------------------- |
| Display | DINCond-Bold / Arial Black | 42–50px | 900  | 截图页公里数大号展示 |
| H1      | 系统字体                   | 22px     | 700  | 页面主标题           |
| H2      | 系统字体                   | 18px     | 600  | 卡片标题             |
| H3      | 系统字体                   | 15px     | 600  | 区块标题             |
| Body    | 系统字体                   | 14px     | 400  | 正文                 |
| Caption | 系统字体                   | 12px     | 400  | 辅助文字             |
| Small   | 系统字体                   | 11px     | 400  | 标签/时间戳          |

### 3. ✨ 微交互体系

| 场景       | 效果                            | 曲线     | 时长  |
| ---------- | ------------------------------- | -------- | ----- |
| 按钮按压   | `scale(0.97)` + 背景加深      | ease     | 100ms |
| 按钮悬停   | 背景色过渡到 hover 色           | ease     | 150ms |
| 卡片 hover | `translateY(-2px)` + 阴影提升 | ease-out | 200ms |
| 列表入场   | 逐项渐入，每项延迟 50ms         | ease-out | 300ms |
| 页面切换   | `fade + translateY(8px)`      | ease     | 200ms |
| Toast 弹入 | 从顶部滑入 + 淡出               | ease-out | 250ms |
| 箭头 hover | 向右滑动 4px                    | ease     | 150ms |
| 图标 hover | 颜色变化 + 轻微旋转             | ease     | 150ms |

## 三、实施清单

### P0 — 核心视觉提升（预计 45min）

- [ ] **emoji → SVG 图标替换**

  - 📥 → 下载图标 (Heroicons ArrowDownTray)
  - 👁 → 预览图标 (Heroicons Eye)
  - 🔄 → 刷新图标 (Heroicons ArrowPath)
  - 💾 → 保存图标 (Heroicons DocumentArrowDown)
  - 📦 → 批量图标 (Heroicons Squares2x2)
  - 🚀 → 开始图标 (Heroicons Play)
  - 📝 → 笔记图标 (Heroicons PencilSquare)
  - 🏃 → 跑步图标 (Heroicons Running)
- [ ] **卡片 hover 浮动效果**

  - `transform: translateY(-2px)`
  - `box-shadow` 从 sm 级提升到 lg 级
  - 过渡 200ms ease-out
- [ ] **全局 focus-visible 轮廓**

  - 所有 button, a, input, select 加 `focus-visible:outline`
  - 2px 品牌色 + 3px 半透明光晕

### P1 — 交互增强（预计 35min）

- [ ] **按钮按压反馈**

  - 所有 `.btn-primary` / `.btn-ghost` / `.btn-danger`
  - `:active { transform: scale(0.97) }`
- [ ] **列表入场动画**

  - Notes 列表、好友列表、搜索结果的 v-for 项
  - 逐项 `transition-delay` 递增 50ms
  - 从 `opacity:0; transform:translateY(8px)` 到正常
- [ ] **图标 hover 动效**

  - 所有行内 SVG 图标
  - hover 时 `color` 过渡 + 轻微 `rotate(3deg)` 或 `scale(1.1)`

### P2 — 细节品质（预计 45min）

- [ ] **空状态插图**

  - Notes 空态：简约笔记 SVG 插图 + "还没有笔记" + 引导按钮
  - Friends 空态：简约用户 SVG 插图 + "还没有好友"
  - Profile 空态：简约统计 SVG 插图
- [ ] **卡片左侧装饰条**

  - 工具卡片、笔记卡片左侧 3px 品牌色竖条
  - 好友笔记用蓝色竖条，自己笔记用绿色竖条
- [ ] **侧栏毛玻璃效果**

  - `background: rgba(255,255,255,0.85)` + `backdrop-filter: blur(12px)`
  - 暗黑模式对应 `rgba(26,29,39,0.85)`

## 六、UI/UX Pro Max 资源引用

- **Style**: Minimalism + Soft UI Evolution — 适合工具型应用
- **Color**: 保留翡翠绿 `#26c99a`（产业匹配：运动健康/健身）
- **Effects**: 柔和阴影 + 平滑过渡（200–300ms）+ 温和 hover 态
- **Typography**: 系统字体（-apple-system）确保渲染性能
- **Avoid**: 亮霓虹色、生硬动画、AI 紫粉渐变
