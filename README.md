# KEEPro

## [点击进入Demo](https://tool.joytion.cn/keep/)

Keep运动截图生成 v3.1

### 功能特性

1. 修改输入框中的内容后可永久保存信息（IndexedDB）。
2. 点击"保存图片"即可自动生成截图并保存到本地。
3. 调用高德地图API获取实时温度湿度，解放双手。
4. 支持手动绘制 / 自动生成运动轨迹路径，并支持路径变色效果。
5. **批量生成**：一键生成多张不同日期、天气、温度的打卡截图。

### 使用方式

1. 用浏览器打开 `index.html`
2. 修改设置面板中的各项参数
3. 点击"保存图片"下载截图
